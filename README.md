# 가격.kr — 커뮤니티 기반 단축 URL 서비스

한글 키워드로 최저가를 찾아보세요. `만두.가격.kr` → 네이버 쇼핑 최저가 페이지로 바로 이동!

## 사용법

브라우저 주소창에 `{키워드}.가격.kr`을 입력하세요.

## 기여하기

[기여 가이드](CONTRIBUTING.md)를 참고해 주세요.

## 기술 스택

- **리다이렉트 엔진:** Cloudflare Workers + KV
- **웹 앱:** Next.js (Vercel)
- **자동화:** GitHub Actions
- **데이터:** GitHub 저장소 (JSON 파일)

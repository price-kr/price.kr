# Design: Tier 1 "가격" 재해석 키워드 확장

## 목표

현재 ~80개인 Tier 1 위트형 키워드를 ~150개로 확장. "○○의 가격"을 창의적으로 재해석하여 사용자에게 "딱 맞는 곳이네!" 경험을 제공.

## 범위

- 50~100개의 새 Tier 1 키워드-URL 매핑 추가
- 생활 밀착형 + 추상적 개념형 복합 구성
- 모든 URL은 웹검색으로 실존/정확성 검증

## 카테고리

### A. 생활 밀착형 "가격" 재해석

일상에서 "이거 얼마야?"라고 물을 법한 상황들.

**인생 이벤트 비용:**
- 결혼, 출산, 이사, 장례, 이혼, 입양, 돌잔치비용
- "결혼.가격.kr" → 결혼비용 통계/계산기

**의료/미용 시술비:**
- 치과, 교정, 라식, 성형, 타투, 필러, 보톡스, 임플란트, 탈모치료
- "라식.가격.kr" → 라식 가격 비교 사이트

**자격증/시험 응시료:**
- 운전면허, 한국사, 자격증, 공인중개사, 변호사
- "운전면허.가격.kr" → 도로교통공단 응시료 안내

**공공요금/세금:**
- 군인월급, 국회의원연봉, 과태료, 주차위반, 음주운전벌금
- "군대.가격.kr" → 병사 봉급표

**반려동물:**
- 강아지입양, 고양이입양, 동물병원
- "동물병원.가격.kr" → 동물의료비 가이드

### B. 추상적 개념 "가격" 재해석

"이것의 가격이 있다면?"이라는 위트.

**시간/노동:**
- 시간(최저시급), 야근(야근수당), 알바(시급검색), 퇴직(퇴직금계산기)
- "시간.가격.kr" → 최저임금위원회

**감정/가치:**
- 행복(세계행복지수), 자유(이민비용), 꿈(창업비용), 사랑(결혼비용)
- "행복.가격.kr" → 세계행복지수 리포트

**법적 대가:**
- 실수(과태료), 거짓말(사기죄), 명예(명예훼손벌금), 비밀(NDA위약금)
- "실수.가격.kr" → 경찰청 범칙금 조회

**한국 문화:**
- 설날(세뱃돈시세), 추석(선물세트가격), 수능(응시료+교재비), 축의금(축의금시세)
- "세뱃돈.가격.kr" → 세뱃돈 시세 통계

## 구현

### 변경 파일
- `scripts/generate-top1000-tsv.ts` — `TIER1_KEYWORDS` Record에 50~100개 추가
- `data/whitelist.json` — 필요시 새 도메인 추가
- `data/**/*.json` — 새 키워드 JSON 파일 생성 (seed-data.ts 경유)
- `scripts/top1000-keywords.tsv` — --force로 재생성
- `docs/DEV_PROGRESS.md`, `docs/DEV_LOG.md` — 업데이트

### URL 선정 기준
1. "이 키워드의 가격을 알고 싶어서 왔다"는 사용자 의도에 가장 부합하는 사이트
2. 직접 가격/비용 정보를 보여주는 페이지 (홈페이지가 아닌 해당 페이지 직링크)
3. 한국어 사이트 우선
4. 정부/공공기관 > 전문비교사이트 > 포털검색

### URL 검증
- 모든 URL을 웹검색으로 실존 여부 확인
- whitelist.json 도메인 추가 시 신뢰할 수 있는 사이트만

### 품질 기준
- blocklist/profanity 통과 필수
- "가격" 재해석이 억지스럽지 않을 것 (자연스러운 연결)
- URL이 실제로 가격/비용 정보를 보여줄 것

### 파이프라인
```bash
# 1. 스크립트에 키워드 추가 후
npx tsx scripts/generate-top1000-tsv.ts --force > scripts/top1000-keywords.tsv
# 2. JSON 파일 생성
npx tsx scripts/seed-data.ts scripts/top1000-keywords.tsv ./data
# 3. 테스트
npm test
```
